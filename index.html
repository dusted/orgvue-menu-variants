Orgvue Menu Variants

Mega menu design explorations for the Orgvue website. Each variant applies a different navigation pattern to the same content structure.

Based on existing sites

View variant Monzo Style Two-panel split layout White left nav pane with muted links + tinted right pane with landmark cards. Contained width, 16px radius. Creative explorations

Users are never overwhelmed — drill into only what they care about.
